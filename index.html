NLP / Table Fact-Checking

T-REX: Table ‚Äì Refute or Entail eXplainer

T-REX is a live, interactive tool for verifying whether a textual claim is supported or contradicted by tabular data using state-of-the-art language models. It highlights relevant cells and streams reasoning steps in real-time to ensure transparency and accessibility.

This project is an advanced continuation of a course project from Language Models and Structured Data, where we explored early-stage table fact-checking systems. The paper has been accepted for publication at ECML PKDD 2025!

üöÄ Original Prototype: Before T-REX became a published tool, I built an early version using transformer.js to perform table fact-checking directly in the browser. This prototype demonstrated the feasibility of client-side AI inference for table reasoning and laid the groundwork for the current system.

NLP / ML

Sub-event Detection in Twitter Streams

This project was developed as part of the Machine and Deep Learning Data Challenge (CSC_51054_EP) at IP Paris. Our goal was to detect sub-events (like goals or red cards) from tweet activity during FIFA World Cup matches.

Approach: We implemented and compared three different strategies: traditional ML models (e.g., LightGBM, SVM), graph-based methods analyzing adjacency matrices, and LLM-based approaches. Our final model, the EnhancedPeriodClassifier, enriched BERTweet embeddings with handcrafted features and reached an accuracy of 77%.

Key Features:

  • Preprocessing pipeline adapted from SOTA research
  • Graph construction and deviation analysis
  • Custom neural architecture combining pre-computed embeddings and multi-head attention
  • Extensive hyperparameter tuning with Optuna
